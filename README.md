# Farmer Voice Assistant

**Description:**  
Telugu voice assistant for farmers using Python. This app helps farmers get **weather updates** and **crop prices** using voice commands.

## Features
- Voice input in Telugu
- Voice output in Telugu
- Provides current weather updates
- Gives random crop price information

## Technologies Used
- Python 3.11
- Libraries: speech_recognition, gTTS, requests
- Runs on Windows PC

## How to Run
1. Open terminal in the project folder
2. Run: python farmer_assistant_final.py
3. Speak your command in Telugu
4. The assistant will respond with the requested information
